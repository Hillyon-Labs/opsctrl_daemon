OpsCtrl

Helm Charts

Official Helm chart repository for OpsCtrl - Kubernetes Pod monitoring and failure detection with automated diagnosis

Quick Start

# Add the repository helm repo add opsctrl https://charts.opsctrl.dev helm repo update # Install with required configuration helm install opsctrl-daemon opsctrl/opsctrl-daemon \ --set clusterRegistration.clusterName="my-cluster" \ --set clusterRegistration.userEmail="you@example.com" \ --set monitoring.watchNamespaces="default,my-app" \ --set secrets.backendUrl="https://api.opsctrl.io/api/v1"

Required: clusterName, userEmail, watchNamespaces, and backendUrl

Configuration

clusterRegistration.clusterName Unique identifier for your cluster
clusterRegistration.userEmail Email for cluster registration
monitoring.watchNamespaces Comma-separated namespaces to monitor
secrets.backendUrl OpsCtrl backend API URL
secrets.webhookUrl Slack webhook for alerts (optional)
